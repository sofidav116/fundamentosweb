LABORATORIO TOTITO - EXTRA

Descripción:
Este laboratorio consiste en el desarrollo de una aplicación web interactiva del juego “Totito” utilizando React. Se implementó la lógica del juego mediante componentes reutilizables y manejo de estado, permitiendo una experiencia dinámica para el usuario. Además, se trabajó un diseño visual moderno con un estilo futurista y estético.

Características:
- Tablero interactivo de 3x3.
- Alternancia automática de turnos entre jugadores (X y O).
- Detección automática de ganador.
- Mensaje de empate cuando el tablero está lleno.
- Historial de movimientos con opción de regresar a jugadas anteriores.
- Botón para reiniciar la partida.
- Diseño visual futurista con colores, sombras y tipografía moderna.

Archivos modificados:
- totito/src/App.js → Contiene la lógica del juego, componentes y manejo del estado.
- totito/src/styles.css → Contiene los estilos visuales y diseño de la interfaz.
